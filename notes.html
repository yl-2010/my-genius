Notes

My notes about this project.

The hardest part about this project was making the navbar/topbar. Every single element was very finicky, and it was hard to balance px size and % size, because I had to use both in different senarios and they don't interact with each other very well.

I included almost all of the elements from the original screenshot from my computer [see other tab for Arc for Windows screenshot]. I only left out a few buttons that weren't applicable in a website senario, like the extensions button. Every button that I did include does work, even the forward, backward, and reload buttons. Try them out!

In addition, every tab in the sidebar is useful and has information in it, so be sure to explore every page!

This website is not designed for scrolling, due to the finnicky nature of the elements in the gray area. If text is ever outside of your view, use ctrl - to zoom out and bring all the text into view.

Finally, I've been having trouble with the embedded YT video on the last tab. It might work, but don't be surprised if it doesn't.
